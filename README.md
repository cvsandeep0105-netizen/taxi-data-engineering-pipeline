# Taxi Data Engineering & Analysis Pipeline

## Project Overview
This project demonstrates a simple data engineering pipeline and exploratory data analysis using taxi trip data.

The project includes data cleaning, transformation, visualization, and analysis of taxi trip patterns.


## Technologies Used

- Python
- Pandas
- Matplotlib
- SQL
- Jupyter Notebook
- Git & GitHub


## Project Structure

data/
- raw taxi dataset
- processed cleaned dataset

notebooks/
- data pipeline notebook
- data analysis notebook

scripts/
- python data processing script

sql/
- SQL analysis queries

visuals/
- charts and visualizations generated during analysis



## Key Insights

- Most taxi trips are short-distance rides.
- Taxi fares increase as trip distance increases.
- Fare values vary across boroughs.
- Travel demand differs between boroughs.
- Payment methods show different usage patterns.



## Project Conclusion

Taxi trip data analysis reveals clear patterns in urban travel behavior.

Short trips dominate taxi usage, while longer trips significantly increase fare values.  
The strong positive relationship between distance and fare confirms expected pricing behavior in taxi services.

Borough-level analysis highlights variations in travel demand across the city.
